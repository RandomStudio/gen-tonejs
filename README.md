
# Generative Audio Toolkit

Some tools for generative audio in the browser, using [ToneJS](https://tonejs.github.io/), ES6 via webpack, and a lightweight custom UI.

## Demo
Go to https://anselanza.github.io/gen-tonejs/

## Setup


### Installation

```
npm install
```

### Start Dev Server

```
npm start
```

### Build Prod Version

```
npm run build
```
